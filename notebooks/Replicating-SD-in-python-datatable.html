Summary#

Weâ€™ve seen how to replicate .SD in datatable. There are other functionalities in .SD that are not presently possible in Pythonâ€™s datatable. It is possible that in the future, .SD will be implemented to allow for custom aggregation functions. That would be truly awesome, as it would allow numpy functions and functions from other Python libraries into datatable.
